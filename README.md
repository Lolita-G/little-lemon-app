Welcome to the Little Lemon Restaurant website! This project is part of the Meta Frontend Developer Capstone Project. It's built with **React** and allows users to explore the restaurant's menu and make reservations online.
## Features
- Browse restaurant menu and specials
- Book a table online
- Form validation for booking
- Responsive design for mobile and desktop
- Accessible UI with ARIA labels and semantic HTML
- Smooth scrolling between sections
## Installation
1. Clone the repository:  
```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git```
2. Navigate to the project directory:  
```bash
cd YOUR_REPO_NAME```
3. Install dependencies:  
```bash
npm install```
4. Start the development server:  
```bash
npm start```
Open [http://localhost:3000](http://localhost:3000) to view the app in your browser.
## Usage
- Navigate through the homepage to view specials, testimonials, and about sections.
- Click on "Book a Table" to access the reservation form.
- Fill in all required fields and submit to make a reservation.
## Technologies Used
- React
- React Router
- JavaScript (ES6+)
